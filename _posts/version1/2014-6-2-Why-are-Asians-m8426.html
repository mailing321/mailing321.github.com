  1. 在自我选择： < / b>来横渡大西洋或太平洋和redtape是美国签证制度的迷宫，你需要有足够的资源和决心< / LI> <礼> 已饥饿： < / B>把自己放在一个贫穷的移民谁蒙受的一切他/她的生活劳苦白费的位置。最后，你看到的“乐土” 。你会让你的新机会使用远比谁一起成长的财富和人民更好因此拒绝见他们已经得到了祝福。这是在第19和贫困人口如何爱尔兰，意大利和犹太移民20世纪初爬上价值链。同时与中国，韩国，尼日利亚，印度移民的新作物。

    < / LI>

  2. 网络： < / b>如果你是孤独和歧视在一个未知的土地上，你往往会形成强大的网络的人喜欢自己。这就是所有的意大利城镇和唐人街怎么来的。如果你想和别人一样在网络中（比如成为医生，工程师，教授或投资银行家），你可以利用两者的连接和知识存在于网络中。这给亚洲及在其他大多数模型少数民族略占优势，在几个领域。在另一面，如果你想成为一个印度足球运动员还是中国时装设计师的好运气。

    < / LI>

  3. 灵活性： < / b>如果你问一个印度人或中国人从东海岸搬到西海岸，反之亦然，他/她会做而不不动声色。事实上，我已经海岸之间移动半打倍。如果您已到达万英里从国土，这是毫无差别旅行几千多英里寻找新的合资企业。因此，你住在它提供的机会最多的地方（这就是为什么你看到在纽约，顺丰等多个少数民族的发展） 。在另一方面，大多数的公民，美国人是通过家庭关系并列，投资家等，并不总是能够移动到适合自己的技能最好的地方。

    < / LI>

  4. 值： < / B>不同的文化把重点放在不同的事情。亚洲＆犹太文化在历史上往往更加重视科学和数学。幸运的是，这些是赚最多的钱对于现在的事情。相比之下，白种美国人倾向于把稍微更强调冒险和运动。 < / LI> < / OL >
